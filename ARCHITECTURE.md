```
1. User Action
   â””â”€> User fills out form on /contact page
       - Name: "John Doe"

       - City: "Mumbai"
       - Role: "Parent"
       - Type: "subscribe"

2. Client-Side Validation
   â””â”€> SubscriptionForm.tsx validates:
       âœ“ All required fields filled
       âœ“ Email format is valid
       âœ“ No empty strings

3. Form Submission
   â””â”€> Sends POST request to /api/subscribe
       {
         "name": "John Doe",
         "email": "john@example.com",
         "city": "Mumbai",
         "role": "parent",
         "type": "subscribe"
       }

   â””â”€> /api/subscribe/route.ts:
       a) Validates data again (security)

       e) Calls sendToGoogleSheets()

5. Data Storage
   â””â”€> Google Sheets Apps Script:
       a) Receives POST request
       b) Parses JSON data
       c) Appends row to sheet:
          [2025-12-07 10:30:00, John Doe, john@example.com, ...]
       d) Returns success response

6. User Feedback
   â””â”€> Success message displayed:
       "Successfully subscribed!"
```
